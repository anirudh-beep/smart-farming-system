Getting Started

How to use FarmX:

  1. Set Your Location: Use GPS detection or manually select your location from the dropdown menus
  2. Analyze Your Soil: Get automatic soil analysis based on your location or input manual soil data
  3. Check Weather: View current weather and 7-day forecast with farming insights
  4. Get Crop Recommendations: Receive AI-powered crop suggestions based on your soil and weather data

Location Detection

GPS Detection:

Click "Detect My Location" to automatically find your coordinates. Make sure to allow location access when prompted.

Manual Selection:

If GPS fails, select your location manually:

  • Choose your country (India)
  • Select your state
  • Pick your district
  • Optionally select your village

Soil Analysis

Auto-Detection:

FarmX automatically analyzes soil based on your location using regional soil data.

Manual Input:

You can override auto-detected values by entering:

  • Soil type (Black Cotton, Red Laterite, Alluvial, etc.)
  • pH level (4.0 - 10.0)
  • Nutrient levels (Nitrogen, Phosphorus, Potassium)

Weather Features

Weather Forecast:

  • Current weather conditions
  • 7-day detailed forecast
  • Farming-specific insights and recommendations

Seasonal Analysis:

  • Best planting times for different crops
  • Risk assessment (drought, flood, temperature, disease)
  • Agricultural calendar with seasonal guidance

Crop Recommendations

AI Recommendations:

Get personalized crop suggestions based on:

  • Your soil type and nutrient levels
  • Local weather patterns
  • Regional agricultural data
  • Market prices and profitability

Fertilizer Guidance:

Enter your budget to get fertilizer recommendations tailored to your crops and soil needs.

About FarmX

FarmX is a comprehensive smart farming platform designed to help farmers make informed decisions about their crops. Our platform combines location-based soil analysis, weather forecasting, and AI-powered crop recommendations to maximize agricultural productivity.

Platform Features

Location Detection

Accurate GPS-based location detection with manual fallback options covering all Indian states and districts.

Soil Analysis

Comprehensive soil type identification, pH analysis, and nutrient level assessment based on regional data.

Weather Integration

Real-time weather data, 7-day forecasts, and seasonal analysis with farming-specific insights.

Crop Recommendations

AI-powered crop suggestions based on soil conditions, weather patterns, and market data.

Fertilizer Guidance

Budget-based fertilizer recommendations with application rates and timing suggestions.

Agricultural Database

Extensive database of 1000+ agricultural records covering various crops, soil types, and regional conditions.

Data Sources

  • Weather Data: WeatherAPI for accurate, real-time weather information
  • Soil Data: Regional soil databases and agricultural research data
  • Crop Data: Agricultural extension services and farming best practices
  • AI Insights: Google Gemini AI for intelligent crop recommendations

Privacy & Security

Your location and farming data are processed securely. We don't store personal location information permanently and use data only to provide accurate agricultural recommendations.

Browser Compatibility

FarmX works on all modern browsers including Chrome, Firefox, Safari, and Edge. For best experience, ensure JavaScript is enabled and allow location access when prompted.

Troubleshooting

Location Detection Issues:

  • Ensure location services are enabled in your browser
  • Check if you're blocking location access
  • Use manual location selection as a fallback

Weather Data Not Loading:

  • Check your internet connection
  • Refresh the page and try again
  • Ensure location is set correctly

Crop Recommendations Not Showing:

  • Complete location and soil analysis first
  • Ensure all required data is filled
  • Check if weather data is loaded
